Unfortunately this means that you must declare your entire svg in the HTML or you must declare your event listener inside of the .svg file as an inline event listener (which is not the preffered practice).

To make Canada clickable we just add a class to all of the paths that have a red fill. Then in our script we use a query selector to gather all these paths into an HTML collection and we apply the event listener to each element in the collection.
